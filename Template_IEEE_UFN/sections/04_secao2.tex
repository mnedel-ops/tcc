\section{Trabalhos correlatos}
Nesta seção, realiza-se uma análise comparativa de jogos sérios que utilizam mecânicas lúdicas para o ensino de Química. A investigação foca em objetivos pedagógicos, tecnologias empregadas e resultados alcançados, estabelecendo as bases para a diferenciação técnica do projeto Alchemon.

\subbsection{ChemCaper: Act I - Petticles in Peril}
O ChemCaper é um jogo sério de interpretação de papéis (Role-Playing Game - RPG) de aventura, desenvolvido pela ACE EdVenture Studio. O objetivo central deste jogo sério é auxiliar estudantes na compreensão de conceitos do currículo IGCSE de Química, abrangendo o uso de aparelhos laboratoriais, técnicas de separação, periodicidade e ligações químicas. Disponível na plataforma Steam, o software utiliza mecanismos de aprendizagem implícita para engajar o público jovem.

\subsection{ChemCraft}
O ChemCraft é um jogo sério de aventura 2D focado no ensino de Química Orgânica de nível superior. O projeto investigou a transposição de sistemas químicos reais para regras de sistemas digitais. As mecânicas de gameplay seguem reações químicas reais, onde as propriedades dos objetos do jogo referenciam valores de dados químicos reais, como massa atômica e energia de dissociação.

\subsection{Batalha QuímiCard}
Desenvolvido para o cenário brasileiro, o Batalha QuímiCard é um jogo sério de cartas digital para a plataforma Android. O objetivo do software é validar o uso de mecânicas de duelo para o ensino de ácidos e bases, focando em forças relativas e constantes de equilíbrio ($K_a$ e $K_b$). A proposta fundamenta-se na aprendizagem tangencial, onde o aluno internaliza o conhecimento científico para obter vantagens estratégicas.

\subsection {Inferência sobre os Trabalhos Correlatos}
A análise evidencia que o uso de mecânicas de RPG e colecionismo em jogos sérios é uma estratégia robusta para mitigar a percepção de abstração da Química. Entretanto, identifica-se uma lacuna estética que o Alchemon pretende preencher: ao adotar uma estética de criaturas única para cada elemento, o projeto visa facilitar a memorização visual e a ancoragem substantiva (TAS). O Alchemon consolida-se como um jogo sério que une uma fundação teórica embasada ao ensino da Tabela Periódica no Ensino Médio.
